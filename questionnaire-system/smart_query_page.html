API接口文档

1. 智能查询接口
POST /api/smart-query/query

功能：使用自然语言查询符合条件的数字人

请求体：

{
  "query": "25-35岁喜欢奢侈品的北京女性"
}

响应示例：

{
  "success": true,
  "query": "25-35岁喜欢奢侈品的北京女性",
  "parsed_query": {
    "structured": {
      "gender": "女",
      "age_min": 25,
      "age_max": 35,
      "residence": "北京"
    },
    "semantic": "喜欢奢侈品 高端品牌"
  },
  "results": [...],
  "count": 15
}
2. 获取查询示例
GET /api/smart-query/examples

功能：获取预设的查询示例

响应示例：

{
  "success": true,
  "examples": [
    {
      "title": "基础人口统计查询",
      "examples": ["找一些25-35岁的女性", "需要北京的男性用户"]
    }
  ]
}
3. 系统状态接口
GET /api/smart-query/status

功能：获取智能查询系统的运行状态

响应示例：

{
  "available": true,
  "engine": {
    "mysql_connected": true,
    "lance_db_connected": true,
    "deepseek_connected": true,
    "vector_count": 29
  },
  "sync": {
    "auto_sync_enabled": true,
    "sync_interval_hours": 24,
    "is_running": true
  }
}
4. 手动同步接口
POST /api/smart-query/sync/manual

功能：手动触发数据同步

请求体：

{
  "type": "full"  // "full" 或 "incremental"
}
5. 同步配置接口
GET /api/smart-query/sync/config
POST /api/smart-query/sync/config

功能：获取或更新数据同步配置

POST请求体：

{
  "interval_hours": 24,
  "auto_sync": true,
  "sync_time": "02:00"
}
